# Student Performance Analysis & Risk Prediction

## Overview
This project analyzes student performance using statistical methods and machine learning techniques to identify at-risk students.

## Dataset
- 45 student records
- Features: Q1–Q12 (survey questions), GPA

## Key Steps
- Data preprocessing and cleaning
- Feature engineering (GPA → Risk classification)
- Hypothesis testing using one-sample t-test
- Data visualization (distribution analysis)
- Machine learning model (Random Forest)

## Results
- Achieved ~80% accuracy
- Identified student risk levels (Low/High)

## Tools & Technologies
- Python (Pandas, NumPy)
- Scikit-learn
- Matplotlib
- Google Colab
